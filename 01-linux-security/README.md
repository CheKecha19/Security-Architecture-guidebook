# Linux Security

## Описание раздела

Linux — основа большинства серверных инфраструктур. Понимание механизмов безопасности Linux критично для:
- Настройки безопасных серверов
- Контейнерной безопасности
- Compliance (CIS Benchmarks)
- Troubleshooting инцидентов

## Ключевые темы

| Тема | Что изучить |
|------|-------------|
| **Landlock** | Песочницы без root |
| **AppArmor** | Profile-based MAC |
| **SELinux** | Label-based MAC |
| **Namespaces** | Изоляция процессов |
| **Seccomp** | Фильтрация системных вызовов |
| **POSIX ACL** | Расширенные права |
| **Capabilities** | Fine-grained privileges |
| **PAM** | Аутентификация |
| **chroot** | Изоляция файловой системы |
| **PolicyKit** | Авторизация действий |
