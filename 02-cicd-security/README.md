# CI/CD Security

## Описание раздела

CI/CD — сердце современной разработки. Безопасность конвейера = безопасность продукта.

## Ключевые темы

| Тема | Что изучить |
|------|-------------|
| **SAST** | Статический анализ кода |
| **DAST** | Динамический анализ |
| **SCA** | Анализ зависимостей |
| **IAST** | Интерактивный анализ |
| **RASP** | Runtime защита |
| **Secrets Management** | Хранение секретов |
| **Supply Chain Security** | Безопасность цепочки |
| **Pipeline Security** | Безопасность CI/CD |
| **DevSecOps** | Интеграция безопасности |
| **Artifact Security** | Подпись артефактов |
