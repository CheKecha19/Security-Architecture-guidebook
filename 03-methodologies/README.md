# Методологии threat modeling и оценки рисков

## Описание раздела

Методологии — это инструменты мышления для анализа угроз и оценки рисков. Они помогают системно подходить к безопасности, не упускать важные аспекты и принимать обоснованные решения.

## Ключевые темы

| # | Тема | Что изучить |
|---|------|-------------|
| 01 | **STRIDE** | Классификация угроз: Spoofing, Tampering, Repudiation, Information Disclosure, DoS, Elevation |
| 02 | **DREAD** | Оценка рисков: Damage, Reproducibility, Exploitability, Affected Users, Discoverability |
| 03 | **PASTA** | Risk-centric подход: 7 этапов от бизнес-анализа до стоимости митигации |
| 04 | **Kill Chain** | 7 этапов атаки: от разведки до достижения цели |
| 05 | **Attack Trees** | Иерархические деревья атак с AND/OR узлами и количественной оценкой |
| 06 | **MITRE ATT&CK** | Матрица тактик и техник adversary |
| 07 | **OCTAVE** | Операционно-ориентированный подход: профили активов, уязвимости, стратегия |
| 08 | **CVSS** | Оценка серьёзности уязвимостей: 0-10 |
| 09 | **LINDDUN** | Privacy-focused: GDPR/152-ФЗ, 7 категорий угроз приватности |
| 10 | **OWASP Threat Modeling** | 4 шага: scope, DFD, угрозы, митигация |
| 11 | **VAST** | Visual Agile Simple Threat modeling для agile-команд |
| 12 | **Trike** | Risk-based подход: акторы, активы, риски |
| 13 | **Diamond Model** | 4 вершины: adversary, infrastructure, capability, victim |
| 14 | **NIST SP 800-154** | Data-Centric Threat Modeling для федеральных систем |
