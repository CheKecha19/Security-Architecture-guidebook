# Device Code Flow

## Что такое Device Code Flow?

Представь, что ты хочешь войти в аккаунт на телевизоре. У телевизора нет клавиатуры. Вводить пароль — боль. Вместо этого телевизор показывает код: "ABCD-1234". Ты берёшь телефон, открываешь сайт, вводишь код. Подтверждаешь. Телевизор входит.

**Device Code Flow** — это именно такой процесс. Устройство без удобного ввода получает код. Пользователь подтверждает на другом устройстве.

## Процесс

| Шаг | Устройство | Пользователь |
|-----|------------|--------------|
| 1 | Запрашивает device_code | — |
| 2 | Показывает user_code | — |
| 3 | — | Вводит user_code на другом устройстве |
| 4 | — | Авторизуется и соглашается |
| 5 | Опрос на получение токена | — |
| 6 | Получает токен | — |

## Параметры

| Параметр | Описание |
|----------|----------|
| client_id | ID приложения |
| device_code | Код для устройства |
| user_code | Код для пользователя |
| verification_uri | Где ввести код |
| expires_in | Срок действия |
| interval | Интервал опроса |

## Когда использовать

| Устройство | Почему |
|------------|--------|
| ТВ | Нет клавиатуры |
| Консоль | Неудобный ввод |
| IoT | Ограниченный интерфейс |
| Принтер | Нет браузера |
