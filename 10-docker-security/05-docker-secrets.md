# Управление секретами в Docker

## Проблема

| Что | Почему плохо |
|-----|--------------|
| ENV | Видно в docker inspect |
| ARG | Видно в истории образа |
| Файл | Может попасть в образ |
| Код | В git |

## Docker Secrets

| Параметр | Описание |
|----------|----------|
| Создание | echo "password" | docker secret create db_password - |
| Использование | --secret db_password |
| Хранение | Зашифрованно в swarm |
| Доступ | Только нужным сервисам |

## HashiCorp Vault

| Параметр | Описание |
|----------|----------|
| Динамические | Временные учётки |
| Ротация | Автоматическая |
| Audit | Логи доступа |
| Интеграция | Docker, Kubernetes |

## Лучшие практики

| Практика | Описание |
|----------|----------|
| Docker Secrets | Для swarm |
| Vault | Для сложных случаев |
| Не ENV | Для паролей |
| Read-only | Файл secrets |
| Ротация | Регулярная смена |
