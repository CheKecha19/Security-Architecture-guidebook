# Безопасность инференса

## Что такое inference?

Представь библиотекаря, который отвечает на вопросы. Каждый вопрос — это обращение. Библиотекарь думает, отвечает. Это и есть инференс.

**Inference** — процесс, когда LLM генерирует ответ на запрос. Это рабочий процесс. Нужно защищать.

## Угрозы

| Угроза | Описание |
|--------|----------|
| Prompt injection | Вредоносный промпт |
| Denial of service | Перегрузка |
| Model stealing | Кража модели |
| Side-channel | Утечка через время/память |

## Защита

| Метод | Описание |
|-------|----------|
| Rate limiting | Ограничение |
| Input validation | Проверка |
| Output filtering | Фильтрация |
| Monitoring | Аномалии |
| Encryption | HTTPS |
