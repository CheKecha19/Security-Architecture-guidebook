# Безопасность цепочки LLM

## Что такое supply chain?

Представь ресторан. В ресторане блюдо готовится из ингредиентов. Ингредиенты привозят поставщики. Если поставщик привезёт испорченное мясо — все клиенты отравятся. Нужно проверять каждого поставщика.

**Supply chain LLM** — это именно такая цепочка. Модель зависит от: данных, библиотек, моделей, инфраструктуры. Каждый компонент — потенциальная угроза.

## Компоненты

| Компонент | Угроза |
|-----------|--------|
| Данные | Poisoning |
| Библиотеки | Уязвимости |
| Модели | Backdoor |
| Инфраструктура | Доступ |
| API | Injection |

## Защита

| Метод | Описание |
|-------|----------|
| SBOM | Перечень компонентов |
| Signing | Подпись моделей |
| Scanning | Проверка уязвимостей |
| Audit | Проверка |
