# Model Poisoning

## Что такое model poisoning?

Представь, что ты учишь ребёнка готовить. Показываешь рецепты. Но кто-то подменил несколько рецептов: вместо сахара — соль. Ребёнок учится. Потом готовит пирог. Кладёт соль. Результат — ужасен.

**Model poisoning** — это именно такая атака. Злоумышленник добавляет в обучающие данные вредоносные примеры. Модель обучается на них. Потом ведёт себя неправильно.

## Типы

| Тип | Описание |
|-----|----------|
| Data poisoning | Отравление данных |
| Backdoor | Триггер для активации |
| Trojan | Скрытая функция |

## Защита

| Метод | Описание |
|-------|----------|
| Data validation | Проверка источников |
| Anomaly detection | Поиск аномалий |
| Differential privacy | Защита данных |
| Federated learning | Распределённое обучение |
| Audit | Проверка моделей |

## Лучшие практики

| Практика | Описание |
|----------|----------|
| Проверенные данные | Только доверенные источники |
| Валидация | Проверка обучающих данных |
| Мониторинг | Аномалии в поведении |
| Резервные копии | Моделей |
